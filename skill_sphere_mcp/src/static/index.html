SkillSphere MCP Server

Welcome to the SkillSphere Model Context Protocol (MCP) Server. This service provides a standardized API that enables LangChain agents to query and interact with a Neo4j-powered Hypergraph-of-Thought, which contains enriched career records and professional experiences.

Key Features

  • Converts plain-text career records into a structured knowledge graph
  • Enriches data through local LLM gleaning loops
  • Provides Node2Vec embeddings for structural similarity search
  • Offers end-to-end tracing and metrics through OpenTelemetry
  • Delivers ATS-optimized résumé generation capabilities
